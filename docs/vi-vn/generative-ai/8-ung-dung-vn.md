|---|---|---|

## 04 · Cơ chế monetization cụ thể (số thật, không "ROI chung chung")

### YouTube RPM theo niche (2026)
> **RPM ≈ 40-55% của CPM** (YouTube giữ 45%). Số dao động — dùng *khoảng*, không phải số đơn.

| Niche faceless | RPM ước tính |
|---|---|
| **Tài chính/đầu tư** | Cao nhất, ~**$10-30+** |
| **True crime/kể chuyện** | ~$4-12 (RPM thấp, volume lớn) |
| **Động lực/self-help** | ~$5-10 |
| **Tech/tutorial** | ~$8-12 |

> 🇻🇳 RPM tiếng Việt/thị trường VN **thấp hơn** US, nhưng cạnh tranh ít hơn + chi phí sản xuất thấp.

### TikTok Shop affiliate
Hoa hồng phổ biến **10-30%** (open plan), thương lượng cao hơn ở target plan. ⚠️ **Clawback:** hoa hồng bị thu hồi khi khách trả hàng (30-60 ngày sau) — đừng tiêu sớm.

### Etsy / Print-on-demand
Biên lợi nhuận **thực tế ~20-35%** (calculator hay thổi lên 45-55%). Phí: ~12-13% (listing + transaction + payment) + **Offsite Ads 12-15%** (bắt buộc khi >$10K/năm).
Công thức giá: `Giá = (Cost + Lợi nhuận muốn) / (1 − tổng % phí)`.

## 05 · SaaS wrap — wrap API gen thành sản phẩm

Mô hình PhotoAI: **wrap 1 API gen + chọn ngách cụ thể + build audience**. Pipeline tối giản:
```
Landing (ngách rõ) → upload ảnh → gọi Flux/Replicate (Ch7) → trả kết quả → Stripe thu phí
```
Ngách VN-friendly: ảnh visa/hồ sơ, ảnh sản phẩm, ảnh profile, logo. **Break-even ~50 khách @ $20/tháng.** Build audience *trước* product (Pieter Levels có 600K followers trước khi bán).

> Code lab Next.js + Stripe: xem cross-ref từ [stage-2](../stage-2/) + áp pattern API ở [Chương 7](./7-pipeline-tu-dong.md).
