#### 🛠️ Cập nhật Sản phẩm & Công cụ
* **OpenAI - spend controls moi cho ChatGPT Enterprise:** OpenAI cong bo ngay `18/06/2026` bo `credit usage analytics` va `updated spend controls` cho `ChatGPT Enterprise`. `Global Admin Console` nay gom du lieu su dung cua ca `ChatGPT` va `Codex` vao mot man hinh, cho phep xem usage theo `user`, `product`, `model`, va doc cung tap du lieu do qua `Cost API`. Voi doi ngu trien khai AI trong doanh nghiep, thay doi nay quan trong vi no bien chi phi agent/model thanh du lieu van hanh co the kiem soat theo nhom, theo ca nhan, va theo budget. Nguon: [OpenAI - New usage analytics and updated spend controls for enterprises](https://openai.com/index/chatgpt-enterprise-spend-controls/).
* **GitHub Copilot - usage metrics API chi tiet hon cho admin:** GitHub cap nhat ngay `19/06/2026` de `Copilot usage metrics API` tra them truong `ai_credits_used` theo tung user trong bao cao `users-1-day` va `users-28-day`. Du lieu nay duoc mo ta la tong AI credits tieu thu moi ngay cua mot nguoi dung, giup enterprise gan muc tieu thu voi adoption va lap ke hoach usage-based billing thuc te hon. Nguon: [GitHub Changelog - AI credits consumed per user now in the Copilot usage metrics API](https://github.blog/changelog/2026-06-19-ai-credits-consumed-per-user-now-in-the-copilot-usage-metrics-api/).
* **GitHub Copilot Code Review - doc `AGENTS.md` ngay tai root repo:** Tu ngay `18/06/2026`, `Copilot code review` da co the doc `AGENTS.md` o thu muc goc repo va dung cac huong dan lien quan khi tao review feedback. GitHub dong thoi them nut `Request` de goi review tren draft pull request nhanh hon va gom bot timeline events de conversation de doc hon. Tac dong truc tiep la repo co the dua convention va expectation vao chinh file `AGENTS.md`, giam phu thuoc vao prompt tay khi review AI. Nguon: [GitHub Changelog - Copilot code review: AGENTS.md support and UI improvements](https://github.blog/changelog/2026-06-18-copilot-code-review-agents-md-support-and-ui-improvements/).

#### 🔬 Nghiên cứu & Học thuật (Báo cáo nổi bật)
* **Hugging Face - Beyond LoRA:** Hugging Face dang bai ngay `18/06/2026` dat lai cau hoi quen thuoc trong PEFT: co nen mac dinh dung LoRA cho moi bai toan fine-tune hay khong. Bai viet dua ra mot chi so rat manh: trong mau `20,834` model cards tren Hub chi nhac duy nhat mot ky thuat PEFT, co `20,509` truong hop la `LoRA`, tuong duong `98.4%`; nhom tac gia dong thoi nhan manh thu vien `PEFT` hien da co hon `40` ky thuat khac nhau. Thong diep ky thuat quan trong o day la he sinh thai dang co xu huong chon LoRA vi inertia tooling va tutorial, khong nhat thiet vi no luon toi uu cho chat luong, VRAM, hay checkpoint size. Nguon: [Hugging Face - Beyond LoRA](https://huggingface.co/blog/peft-beyond-lora).
