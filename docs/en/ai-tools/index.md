# AI Tools — A Practical Toolkit

<p style="font-size: 48px; line-height: 1; margin: 0 0 12px;">🧰</p>

::: tip 🔥 In 30 seconds
Knowing one “great” AI tool but not *when to use which* still slows you down. This module isn't theory — each chapter is **one tool, one real workflow**: where to install it, how it's priced, what to type, and the traps that bite.
**💸 Why it matters:** picking the right tool for the job saves hours every day and real subscription money.

> **There is no “best tool” — only “the right tool for your job.”**
> This module helps you build a personal *toolkit*: conversational assistants, coding agents, a dev platform, and self-hosted agents.

## 🧭 Suggested path

1. **New to AI** → start with [ChatGPT](./1-chatgpt) (all-round assistant) + [GitHub](./5-github) (foundational skill).
2. **Search & research** → [Perplexity](./10-perplexity) (with sources), [NotebookLM](./13-notebooklm) (from your own docs).
3. **Code with AI** → [Cursor](./8-cursor) (IDE) or [Claude Code](./2-claude-code) / [OpenAI Codex](./4-codex) (terminal/agent).
4. **Non-coders / office work** → [Claude Cowork](./3-claude-cowork); automate with [n8n](./12-n8n).
5. **Compare the big chatbots** → [ChatGPT](./1-chatgpt) vs [Gemini](./9-gemini) vs [Grok](./11-grok).
6. **Your own self-hosted agent** → [OpenClaw](./6-openclaw), [Hermes Agent](./7-hermes-agent).

## 🔗 Combining tools — a real workflow

Real power isn't in one tool — it's in **chaining** them. For example, taking a web-app idea from zero to shipped:

| Step | What you do | Tool |
|---|---|---|
| 1. Idea & spec | Brainstorm, write a PRD, sketch screen flows | [ChatGPT](./1-chatgpt) |
| 2. Build code | Hand over the spec, let the agent scaffold & edit files | [Claude Code](./2-claude-code) or [Codex](./4-codex) |
| 3. Store & collaborate | Commit, branch, Pull Request, review | [GitHub](./5-github) |
| 4. Background automation | An agent runs on a schedule, scans issues, reports back | [Cowork](./3-claude-cowork) / [Hermes](./7-hermes-agent) / [n8n](./12-n8n) |

> 💡 Tip: use **one conversational assistant to THINK** (ChatGPT/Claude) and **one coding agent to BUILD** (Claude Code/Codex) — don't make a single tool carry everything.

## ❓ Quick FAQ

::: details Total beginner — which tool first?
**ChatGPT** (all-round assistant, nothing to install) + **GitHub** (foundational skill). These two cover almost everything and are mostly free.
:::

::: details Don't code, want AI to handle paperwork/research?
[ChatGPT](./1-chatgpt) for one-off tasks; [Claude Cowork](./3-claude-cowork) when you want to *hand over a multi-step goal* and get a finished deliverable.
:::

::: details Want AI to write code for a real project — Claude Code or Codex?
Pick by **the plan you already have**: paying for Claude Pro/Max → [Claude Code](./2-claude-code); paying for ChatGPT → [Codex](./4-codex). Both are strong — the difference matters less than how disciplined you are with them.
:::

::: details Worried about data/privacy, don't want to send things to the cloud?
Use a **self-hosted agent** ([OpenClaw](./6-openclaw) / [Hermes Agent](./7-hermes-agent)) with a local model (Ollama) → your data never leaves your machine.
:::

::: warning ⏱️ A note on timeliness
AI tools change pricing & features **fast**. This content is current as of around **mid-2026** — always check the official site before buying a plan.
:::
