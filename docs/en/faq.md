# Frequently Asked Questions

The most common questions about **Task AI Wiki**. Can't find an answer? See [Contact](./contact) or open an [Issue](https://github.com/MichaelDo0101/learning-ai/issues).

## About the course

::: details What is Task AI Wiki?
An open course teaching **AI Vibe Coding** — using AI to build real software, from zero to shipped. It includes several modules: Stage 0–3, Generative AI, Agentic AI, and an **AI Tools** toolkit (19 tools). Content is free.
:::

::: details Is it free?
**Yes.** All content is free under the CC BY-NC-SA 4.0 license (non-commercial, with attribution, share-alike). You only pay if you subscribe to third-party AI tools (ChatGPT, Cursor…).
:::

::: details Do I need to know how to code first?
**Not required.** [Stage 1](./stage-1/learning-map/) is designed for beginners. If you already code, jump to [Stage 2](./stage-2/frontend/lovart-assets/) / [Stage 3](./stage-3/core-skills/basics/).
:::

## Path & how to learn

::: details Where should I start?
See the [Learning Map](./stage-1/learning-map/) to pick an entry point by level and goal. Beginners: start at Stage 1. Want to use tools right away: go to [AI Tools](./ai-tools/).
:::

::: details How long does it take?
Depends on your goal. Basics + your first prototype: 2–4 weeks of steady study. Going deep on full-stack or agentic AI: a few months. Each module has its own roadmap.
:::

::: details Which AI tool should I use?
Beginners should start with **ChatGPT** + **GitHub**. Details on each tool (pricing, workflow, case studies) are in the [AI Tools](./ai-tools/) module.
:::

## Project & contributing

::: details How is Task AI Wiki different from the original easy-vibe?
Task AI Wiki is **derived** from [datawhalechina/easy-vibe](https://github.com/datawhalechina/easy-vibe) (open-source), fully localized, re-edited for a Vietnamese context, and extended with new modules (Generative AI, Agentic AI, AI Tools).
:::

::: details I want to report a bug / suggest / contribute content?
Open an [Issue](https://github.com/MichaelDo0101/learning-ai/issues) for bugs, use [Discussions](https://github.com/MichaelDo0101/learning-ai/discussions) to talk, or read the [Contribution Guide](https://github.com/MichaelDo0101/learning-ai/blob/main/CONTRIBUTING.md) to send a Pull Request.
:::

::: details Can I reuse content for my class/blog?
Yes, under **CC BY-NC-SA 4.0**: non-commercial, with **attribution** to Task AI Wiki, and share-alike. See the [license](https://github.com/MichaelDo0101/learning-ai/blob/main/LICENSE).
:::
