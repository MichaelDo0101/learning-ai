## 03 Suno v5.5 pipeline — 6 steps

::: tip 🎵 Standard 2026 workflow
```
1. Lyrics ──→ 2. Genre/mood ──→ 3. Suno gen ──→ 4. Refine ──→ 5. Master ──→ 6. Distribute
   (LLM)        (prompt)         (multi take)     (extend/edit)  (LANDR)       (DistroKid)
   30 min       5 min            1-2 hrs          30-60 min      15 min        1 day
```
:::

### Step 1. Lyrics (LLM / human)

**Option A**: write yourself — best storytelling, unique
**Option B**: LLM generates → human edits — fast, still artistic

LLM prompt:
```
Write lyrics in [language] for [genre: bolero/indie/rap] song.
Theme: [theme]
Structure: verse 1 → chorus → verse 2 → chorus → bridge → chorus
Each verse 4-8 lines, chorus 4 lines.
Rhyme scheme: [aabb / abab / free]
```

### Step 2. Genre/mood prompt for Suno

Suno has 2 modes:
- **Simple Mode**: 1 description prompt
- **Custom Mode**: separate lyrics + style

Custom Mode example:
```
Style: K-indie folk, 75 BPM, acoustic guitar + piano + soft pad, 
female mezzo, intimate vocal recording, lo-fi production.
Lyrics: [paste lyrics]
```

### Step 3. Generate (Suno v5.5)

- Each prompt → **2 versions**
- Generate **5-10 times** with prompt variations → pick best
- **Personas / Voices** (v5.5): create voice persona for album consistency

### Step 4. Refine

- **Extend**: lengthen song 2min → 4-8min
- **Replace section**: rewrite specific verse
- **Cover**: convert existing song to new genre
- **Stems**: separate vocal/drum/bass/instrument for free mixing

### Step 5. Master

| Tool | Cost | Use case |
|------|------|------|
| **LANDR** | $4-9/track | Cloud auto-mastering |
| **eMastered** | $10/track | Good for pop |
| **Suno built-in mastering** | $0 (Pro tier) | Sufficient for release |
| **Pro engineer manual** | $50-300/track | When deal is big |

### Step 6. Distribute

| Platform | Cost | Royalty |
|------|------|------|
| **DistroKid** | $23/year | 100% |
| **TuneCore** | $15-50/year/album | 100% |
| **CD Baby** | $9.95/single | 91% |
| **Amuse** | Free tier | 80-100% |

**Process**: Master track + cover art + metadata → DistroKid → Spotify / Apple Music / Amazon / YouTube Music / TikTok / Tidal (~7 days).

## 07 Monetization paths

::: tip 💰 5 ways to monetize

**1. Streaming royalty (Spotify, Apple Music)**
- ~$3-5 / 1,000 streams
- Need consistent release to accumulate
- Aventhis: 3 albums / 4 months → 1M monthly listeners

**2. Sync license (TV, ads, YouTube)**
- $50-5,000 / sync
- Platforms: Musicbed, Artlist, Epidemic
- Genre fit: cinematic, ambient, corporate

**3. Brand deal / commission**
- AI music for ads: $500-5,000 / track
- Karaoke MV for venues: $100-500 / track

**4. Suno Personas as a service**
- Create "voice persona" for other artists
- Charge $200-1,000 / persona setup

**5. Cover production**
- Convert old local songs to new genres
- Sell to cover singers / TikTokers
:::

## 08 Common pitfalls

::: warning 🚨 5 mistakes

**1. Clone celebrity voice** — Spotify bans, can be sued. Use own Persona.

**2. Spam uploads** — 50 tracks/day = flagged. Max 2-3 tracks/week.

**3. Skip mastering** — raw Suno sounds "AI-ish." LANDR $4 fixes 80% of issues.

**4. No custom cover art** — Spotify algo favors good art. Use Midjourney + cover template.

**5. No identity/persona** — releasing 1 track per persona makes fan building hard. Aventhis is "outlaw country" consistently; Xania Monet is "R&B + spiritual."
:::
