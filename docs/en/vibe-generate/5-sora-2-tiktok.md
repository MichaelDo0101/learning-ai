## 06 Prompt pack — Sora 2 / Veo 3.1

::: tip 📝 5 format templates

**1. Cameo crossover (Sora 2)**
```
Selected cameos: [Person A] and [Person B]
Setting: [location detail]
Action: [interaction, 2-3 beats]
Camera: medium shot, slight handheld
Audio: dialogue with [tone], background ambient
Duration: 15s
Style: cinematic, golden hour, natural skin
```

**2. Bodycam POV (Veo 3.1)**
```
First-person POV bodycam footage, [setting]. 
[Subject] approaches camera, says: "[dialogue 1 line]". 
Cam shakes slightly, audio includes ambient noise + voice. 
12s. Realistic, no music.
```

**3. Parody commercial (Sora 2)**
```
Spoof commercial for [fake product name]. 
Spokesperson with exaggerated enthusiasm, cheesy 90s infomercial. 
Includes obvious flaw / absurd benefit. 
20s with voiceover + product shot.
```

**4. Surreal meme (Kling 2.5)**

```
[Animal] performing [human activity] in [setting]. 
Realistic but absurd. Smooth camera, natural lighting, 10s.
```

```
A [nationality] [character] in [local location]. 
[Activity with local detail]. Sound includes [language] 
ambient + optional "[1 line]". 15s. Documentary style, handheld.
```

## 07 Common pitfalls

::: warning 🚨 6 TikTok AI creator mistakes

**1. Clone real celebs without permission** → removed, possible lawsuit (Sora locks cameo if detected)

**2. Outdated format** → each format has 2-4 week shelf life, must spot new trends

**3. English caption when targeting local market** → low reach, algo demotes

**4. Hashtag spam (20+)** → TikTok algo demotes, use 3-5 quality

**5. Skip native trending music** → algo demotes, must use trending sound

**6. Sporadic posts** → need 1-3/day consistency over 30 days for algo "push"
:::
